铁路12306是基于gin开发的售票系统，实现了用户信息注册、余票查询、售票、退票和改签等功能。

## 快速部署
### 安装docker

[安装](https://docs.docker.com/engine/install/)

### 安装docker-compose

[安装](https://docs.docker.com/compose/install/)

### 部署项目

```shell
docker-compose up
```
